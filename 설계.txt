[공통사항]
1. 과제명: AI 기반 경쟁사 동향 모니터링 시스템 구축
2. 목표
 - 반복·수동적인 정보 검색 업무를 자동화하여 업무 효율성 확보
 - 경쟁사 관련 뉴스·보도자료·공시정보를 수집·분석하여 경영진 Reporting 등 의사결정 지원
 - 관세 정책 등 주요 시장 환경 변화에 대한 데이터 기반의 신속한 대응 가능
3. 개발 요구 조건
 - 초보자 학습/개발 병행으로 유지보수가 쉽게 구현
 - PC 웹 및 모바일 UI 구현-웹앱(Flask + HTML 템플릿) 형태, 복잡하지 않고 실용적인 기능 중심으로 개발
 - LLM은 openAI API key 사용
 - 환경 관련 내용은 .env 파일에 적용
4. 기능 구현
 - 전체적인 흐름 : 웹크롤링이나 API를 통해 데이터를 수집하고, 수집한 데이터를 LLM에 보내서 분석하고 웹 데시보드 형태로 출력
 - 수집 데이터 : 네이버 뉴스 API, 경쟁사(동국씨엠, KG스틸, 세아씨엠, DK동신, 아주스틸) 홈페이지의 뉴스룸 게시물, 환율(원/달러, 짯/달러, 짯/원), 전력비(한전 게시 데이터), LNG, LME(Al,Zn), 경쟁사의 다트 공시정보(최근7일)- 실적 공시는 용약정보 츨력
 - UI 구성 : 


* 수집 데이터는 정확히 어떤 데이터인지 웹사이트까지 정리해서 메일로 알려줄것
